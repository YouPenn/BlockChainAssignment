# AppleBox

Steps to Launch AppleBox:


Install MetaMask:
Install the MetaMask extension on your web browser.
Create a new account or log in to an existing one.
Add the "Ethereum Holesky" network to your MetaMask wallet.


Install XAMPP and Apache NetBeans IDE 12.0:
Download and install XAMPP (a local web server environment).
Download and install Apache NetBeans IDE 12.0 (an integrated development environment for working with PHP and other programming languages).


Unzip the Project Files:
Create a folder name like BlockChainAssignment in xampp/htdocs.
Extract the BlockChain.zip to the xampp/htdocs/BlockChainAssignment folder.
The full path should look something like C:\xampp\htdocs\BlockChainAssignment.


Open the Project in NetBeans:
Launch Apache NetBeans IDE 12.0.
Open the project by navigating to "File" than "Open Project" and selecting the BlockChainAssignment folder.


Run the Project:
Once the project is opened in NetBeans, run it.
This should open the user homepage in your default web browser.


Admin Login:
To log in as an admin, use the following credentials:

Admin ID: A001
Admin Password: adminpassword

Admin ID: A002
Admin Password: adminpassword

*You can only edit and delete the products that you have added. You cannot edit or delete products that are owned by other users.
